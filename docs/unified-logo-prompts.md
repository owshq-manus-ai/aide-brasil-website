## Tips for Using These Prompts

### For Lucid Origin & Dynamic Models
1. Include all specifications in one flowing sentence
2. Mention "without" or "no" for unwanted elements naturally in the description
3. Be specific about colors using hex codes
4. Reference known design styles (ChatGPT, Claude) for consistency
5. Emphasize "flat", "2D", "vector" multiple times

### Key Phrases That Work Well
- "Ultra minimalist flat vector design"
- "Professional AI company branding"  
- "Thin elegant lines with consistent stroke weight"
- "Pure white background without textures"
- "Geometric simplicity matching ChatGPT aesthetic"
- "Brazilian metallic gradient from #10b981 to #fbbf24"

### Avoid These Terms
- Don't use "futuristic" (leads to complex designs)
- Don't say "modern" without "minimalist" (gets too elaborate)
- Don't mention "technology" without "simple" (adds unnecessary elements)
- Don't use "innovative" (tends to add extra details)
